# 任意密码修改脚本
## 使用方法如下:
## 1、需要将利用FOFA搜索到的资产或者说想要测试的URL放入到任意的TXT文件中
## 2、通过命令行界面去调用存储着URL的TXT文件
## 3、观察返回的结果,如果显示密码修改成功,且显示出修改之后的密码,说明这个URL中存在这个漏洞
< br >
# 脚本的使用方法演示：
## 此处演示使用的是自己本地搭建的漏洞站点
## 1、将本地搭建的站点存放到TT文件中
## 2、使用python3 PasswordChange.py url.txt进行执行
